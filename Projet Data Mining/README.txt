Le dossier contient les deux projets : Apriori et Apriori parallèle.

Chaque dossier est un projet Netbeans qui contient les codes sources et les jars des deux projets (les JARs sont disponibles dans le dossier dist de chaque projet).
Le dossier APriori contient aussi des fichiers d'exemples (sample.dat = 100 lignes, sample2.dat = 500 lignes, ...)

Les paramètres de l'algorithme apriori standard sont dans le code, pour les changer il faut donc recompiler le projet.

Le JAR d'apriori parallèle doit être exporté dans un environnement Hadoop.
